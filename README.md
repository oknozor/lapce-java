Lapce Java
---
[Lapce](https://lapce.dev/) LSP plugin for java, powered by Eclipse JDT Language Server


## Installation

To install from source use the provided [just](https://github.com/casey/just) recipes:
 
**Lapce stable plugin:**
```shell
❯ just install-stable
```
 
**Lapce debug plugin:**
```shell
❯ just install-debug
```
